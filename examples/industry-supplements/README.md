# Industry Supplements

This directory contains scaffolding for creating new domain-specific supplements.

Currently here:

- **[`template.md`](./template.md)** — A blank scaffold for new domain supplements. Copy this when starting a new industry-specific doc.

## How to Create a New Domain Supplement

1. **Open an issue first** using the "Domain Supplement Proposal" template — this avoids duplicate work and gets community feedback on scope.

2. **Copy the template** to `docs/domain-supplements/[your-industry].md`.

3. **Fill it in** following the structure. Don't repeat universal rules — only override or extend what's in `general.md`.

4. **Get native-speaker review** before submitting if at all possible. Industry-specific Ukrainian often has competing conventions (especially regional vs Kyiv-standard usage).

5. **Submit a PR** with the new supplement plus an entry in the top-level `CHANGELOG.md`.
